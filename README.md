# vkphp
从零开始写一个php框架

# 安装使用
1、搭建php运行环境，apache/nginx + php + mysql。  

2、利用navicat等工具连接到本地mysql数据库，创建数据库vkphp，字符集选择`utf8 -- UTF-8 Unicode`，排序方式选择 `utf8_general_ci`。

3、导入vkphp项目中的vkphp.sql文件。

4、更改app/smarty文件夹权限。
```
cd app/
chmod -R 777 smarty
```
